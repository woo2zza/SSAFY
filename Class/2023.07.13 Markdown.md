# 순서 리스트 적기
1. 1하고.적은 후 한칸 띄우고 써보세요 그리고 엔터
2. 또 적어보세요
3. 엔터를 한번 두번 천천히 눌러 보세요

# 그냥 리스트 적기
순서가 없는 리스트는 - 또는 * 또는 + 기호를 사용합니다.
- 순서가 없는 리스트 연습
- 엔터를 여러번 눌러 보세요
- 또는 shift+tab키를 눌러도 됩니다 (안되면 백스페이스)
